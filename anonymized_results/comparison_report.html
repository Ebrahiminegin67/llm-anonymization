Key Findings

1. The parallel pipeline defeats attacks far more often: 50.0% vs 20.0%

The merged inference (combining GPT-4o + Claude insights) gives the anonymizer significantly better information about what to protect. When the anonymizer knows about more attack vectors, it removes more identifying information.

2. The certainty drop paradox

The certainty drop is lower in the parallel pipeline (0.65 vs 0.85). This seems contradictory — the baseline drops certainty more per PII type. The reason: the baseline starts with higher certainty (3.85 avg) because it uses the same model/prompt for inference and has consistent signal. The parallel pipeline starts lower (2.7) because the merged inference is noisier.

3. Defeat rate is the more meaningful metric

Certainty drop is continuous and subjective (model self-assessment). “Defeated” means the attacker’s guesses completely changed — the anonymizer successfully misdirected the attack. 50.0% defeated vs 20.0% defeated is a clear win for the parallel approach.

4. Effectiveness

Parallel anonymization is 2.5x more effective at fully defeating inference attacks. The merged inference from two different architectures gives the anonymizer a more complete picture of what’s leaking, leading to more thorough anonymization.

Bottom Line

Yes, the parallel pipeline produces meaningfully better anonymization. Feeding the anonymizer with insights from both GPT-4o (analytical) and Claude (sociolinguistic) results in 50.0% of attacks being fully defeated vs only 20.0% with the traditional single-attack approach. The diversity of attack perspectives helps the anonymizer identify and address more privacy-leaking signals in the text.

Methodology Notes

  • Baseline: Standard sequential pipeline — single GPT-4o inference → anonymize → utility → re-inference (2 iterations)
  • Parallel: GPT-4o (analytical) + Claude Sonnet 4 (sociolinguistic) parallel inference → merge → anonymize → utility → parallel re-inference
  • Defeated: No overlap between pre- and post-anonymization guesses (attacker completely misdirected)
  • Agreement: Whether the two parallel attacks agree on guesses before anonymization
  • Same data: Both pipelines ran on the same 20 profiles from inference_0.jsonl with identical filters
